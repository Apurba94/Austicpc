# AUST-ICPC Web Application

A web application created using HTML, CSS, PHP, and MySQL on XAMPP.

## Tech Stack
- HTML, CSS, PHP, MySQL
- Development environment: Visual Studio Code, XAMPP

## Repository
- https://github.com/Apurba94/Austicpc

## How to Run Locally
1. Install XAMPP.
2. Clone this repository into the `htdocs` directory.
3. Start Apache and MySQL from XAMPP.
4. Access the project in your browser via `http://localhost/<folder-name>`.
